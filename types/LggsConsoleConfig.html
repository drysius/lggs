    disable_colors

    disable_colors: boolean

    In some types of hosting, the terminal does not support ansi colors or uses the terminal to display logs. The lggs module uses arguments that apply colors to the terminal using ansi codes, which can make logs difficult to read when saved in .txt files due to the presence of several random characters.

    To solve this problem, this boolean has been added, that, when activated, causes the lggs module to ignore the color codes and imprint simple logs on the terminal, without color formatting.

    Hosts that this boolean becomes useful: [Discloud, Squarecloud]
